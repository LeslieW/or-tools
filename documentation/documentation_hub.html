We are working hard to provide useful documentation. The documentation is distributed in several documents:

  • the Getting started page: explains how to download and install the library.
  • the wiki: contains nice examples. It will gradually be replaced by the user's manual.
  • the user's manual: a user's guide that explains the Constraint Programming part of the library.
  • the tutorial examples: all the complete examples presented in the user's manual.
  • the reference manual: the documentation generated by doxygen.
  • the presentation slides: intended for teaching sessions.
  • the lab sessions: 50 minutes modules.
  • the F.A.Q.: frequently asked questions.

The or-tools documentation is covered by the Apache License 2.0.
