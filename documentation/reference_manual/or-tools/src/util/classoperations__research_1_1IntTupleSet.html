int operations_research::IntTupleSet::Insert ( const std::vector< int > &  tuple) [inline]

Inserts the tuple to the set.

It does nothing if the tuple is already in the set. The size of the tuple must be equal to the arity of the set. It returns the index at which the tuple was inserted (-1 if it was already present).
