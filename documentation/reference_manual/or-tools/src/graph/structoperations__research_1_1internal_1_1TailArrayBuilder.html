template<typename GraphType, bool has_reverse_arcs>
struct operations_research::internal::TailArrayBuilder< GraphType, has_reverse_arcs >

The TailArrayBuilder class template is not expected to be used by clients.

It is a helper for the TailArrayManager template.

The TailArrayBuilder for graphs with reverse arcs does nothing.
