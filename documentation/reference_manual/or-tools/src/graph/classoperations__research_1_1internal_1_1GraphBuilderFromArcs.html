template<typename GraphType, bool is_dynamic>
class operations_research::internal::GraphBuilderFromArcs< GraphType, is_dynamic >

The GraphBuilderFromArcs class template is not expected to be used by clients.

It is a helper for the AnnotatedGraphBuildManager template.

Deletes itself upon returning the graph!
